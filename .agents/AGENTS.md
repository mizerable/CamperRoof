# Testing Requirements

- **ALWAYS run tests autonomously:** Whenever making code changes, you MUST run the test suite yourself (e.g., `pio test`) without asking for permission.
- **Iterate until green:** If the tests fail, you MUST autonomously investigate the failure, fix the code, and re-run the tests. You may only present the result to the user once ALL tests pass successfully.
- **Do not wait for the user to test:** The entire purpose of tests is to allow the agent to verify its own work. Never tell the user to test it themselves if you can run the tests in your own environment.
